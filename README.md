## Overview

**Buhmann_StockStatusSmile** is a bridge package that enables full compatibility between the [buhmann/module-stock-status](https://github.com/buhmann/StockStatus) filter and **Smile ElasticSuite** layered navigation.

This module extends the stock status filter to work seamlessly with ElasticSuite while maintaining all features from the base module:
- Single and multi-select filter modes
- Comma-separated URL format (`?stock_status=0,1`)
- Configurable filter position and title
- Full Elasticsearch/OpenSearch integration

---

## Installation

### Via Composer (Recommended)

```bash
composer require buhmann/module-stock-status-smile
```
